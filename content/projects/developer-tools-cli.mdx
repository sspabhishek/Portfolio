## Overview

Developed **ManageUpgrades**, a production-ready SaaS platform that enables businesses to manage software version releases, user upgrade flows, and customer communications efficiently. 

The platform is built using modern full-stack technologies with scalable AWS infrastructure and secure payment integration.

## The Challenge

Businesses struggle to:
- Track software version releases
- Notify users about updates
- Manage upgrade plans
- Handle subscription billing securely

The challenge was to design a scalable, secure, and automation-driven system that simplifies release and upgrade management while supporting subscription-based monetization.

## Technical Stack

- **Frontend**: Next.js 15, React, Tailwind CSS
- **Authentication**: AWS Cognito (Amplify Auth)
- **Backend**: AWS Lambda (Python)
- **Database**: DynamoDB
- **Payments**: Stripe Subscriptions & Webhooks
- **Cloud Infrastructure**: AWS S3, CloudFront
- **Deployment**: AWS Cloud Hosting

## Key Features

- 🔐 Secure authentication with AWS Cognito
- 📦 Version & release management dashboard
- 💳 Stripe subscription integration with webhook automation
- ⚡ Serverless backend using AWS Lambda
- 🗄 DynamoDB-based subscription tracking
- 🌍 Global deployment using AWS S3 + CloudFront
- 🎨 Modern responsive UI with Tailwind CSS

- Scalable serverless infrastructure
