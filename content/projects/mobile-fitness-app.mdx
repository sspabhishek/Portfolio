## Overview

Led the complete UI/UX revamp of quizwizard.net as part of the Gencoft Technologies internship. Redesigned the user interface for improved engagement and developed AWS Lambda functions for location-based Stripe payment gateway integration.

## The Challenge

The existing quiz platform had outdated UI/UX leading to low user engagement. Additionally, the payment system needed a scalable, serverless solution that could handle location-based pricing and compliance requirements.

## Technical Stack

- **Frontend**: Modern UI/UX redesign with responsive layout
- **Backend**: AWS Lambda for serverless payment processing
- **Payments**: Stripe Payment Gateway with location-based pricing
- **Infrastructure**: AWS API Gateway for secure API management

## Key Features

- Complete UI/UX redesign improving visual appeal and usability
- AWS Lambda functions for serverless payment processing
- Location-based Stripe payment gateway integration
- Improved scalability and compliance through serverless architecture
- Responsive design across all devices

## Results

- **30%** increase in user engagement post-revamp
- Improved scalability through serverless architecture
- Seamless Stripe payment integration with location-based compliance
- Reduced infrastructure costs with AWS Lambda
