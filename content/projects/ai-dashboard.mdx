## Overview

A multiplayer quiz application built with the MERN stack and WebSockets, supporting 100+ concurrent users with real-time leaderboard updates, JWT-based authentication, and persistent sessions.

## The Challenge

Creating an engaging real-time multiplayer quiz experience that handles concurrent users, maintains low latency for leaderboard updates, and provides persistent session management.

## Technical Stack

- **Frontend**: React.js with Tailwind CSS and Redux for state management
- **Backend**: Node.js with Express.js, MongoDB with optimized indexing
- **Real-time**: Socket.io for WebSocket-based multiplayer support
- **Auth**: JWT-based authentication with persistent sessions
- **Deployment**: Backend on Render/Railway with WebSocket support

## Key Features

- Real-time multiplayer quiz sessions supporting 100+ concurrent users
- WebSocket-based live leaderboard updates
- Interactive UI with Tailwind CSS and Redux state management
- JWT-based authentication with persistent sessions
- Optimized MongoDB indexing for fast data retrieval
- Session time increased by 40% through engaging UX

## Results

- **100+** concurrent users supported
- **40%** increase in session time
- **25%** improvement in user engagement
- **35%** reduction in load time through MongoDB indexing optimization
- **18%** increase in user retention
